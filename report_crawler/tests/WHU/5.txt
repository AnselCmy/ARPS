Abstract：
Over years of software development, a vast amount of software engineering data has been generated. These data include source code, bug reports, metrics, console logs, execution traces, etc. Many organizations are now maintaining large softwarerepositories. The open source movement also significantly increases the amount of data from avariety of sources.Various data analytics techniques can be applied to mine the software engineering data with the aim ofimproving software quality and development productivity. Software development, testing, debugging, and maintenance tasks can be also automated. In this talk, I will briefly introduce some of my recent work on data-driven approaches to software engineering.
